# reaper-flake

Declare your Reaper configuration with Nix.

# Packages

- Reaper: 7.74
- ReaPack: 1.2.6
- SWS: 2.14.0.7
- Reapertips Theme 1.90

Packages are copied from nixpkgs with updated hashes.

# Roadmap

Add home manager module that configures nearly everything that is configurable in reaper.

Reaper stores all of it's configuration files in ~/.config/REAPER/ in almost complete plain text format, making it perfect for a home-manager module.
